# aaxclean-cli
Command line interface for [AAXClean](https://github.com/Mbucari/AAXClean)

## Currently Supports:
- AAX and AAXC files
- User-defined chapters
- Input AAX(C) files
  - From local file system
  - From an http(s) url
    - Custom user-agent string
    - User defined web cookies
- Output
  - Single m4b file
    - to local file system
